# AURA Studio v2 Sprint 2

Sprint 2 adds an interactive vehicle and cabin preview to the premium Studio v2 shell.

## Access

```bash
npm run dev --workspace @aura-dcos/studio
```

Open:

```text
http://localhost:5173/studio-v2.html
```

## Deliverables

- Vehicle, Front Cabin and Rear Cabin view switcher.
- Clickable surface preview.
- Surface detail panel for demo talking points.
- Transparent windshield default mode.
- Digital windshield on-demand toggle.
- Front cabin digital dashboard, doors, console and armrest.
- Rear cabin digital roof, full seat backs, rear wall and floor.
- Status rail updates based on selected view.

## Notes

This sprint keeps Studio v1 available at `/` and extends the v2 presentation shell at `/studio-v2.html`.
