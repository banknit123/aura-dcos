# Phase Z — Production Release

Phase Z introduces the production release management seam for AURA DCOS.

## Package

`@aura-dcos/production-release`

## Scope

- Release candidate model.
- Release artifact tracking.
- Release gate tracking.
- Readiness assessment.
- Default gates for typecheck, tests, documentation, safety and cybersecurity.

## Production Notes

This package records and assesses release readiness. It does not replace OEM validation. Real vehicle deployment still requires hardware-specific certification, cybersecurity review, functional safety review, HMI compliance and regional regulatory sign-off.
