# Phase V20 — Vehicle Integration Studio

The `@aura-dcos/vehicle-integration-studio` package provides the model for Studio panels that inspect and simulate vehicle integration.

## Scope

- ECU discovery panel.
- Diagnostics panel.
- Signal Watch panel.
- Command Simulation panel.
- Sensor Streams panel.
- OTA panel.
- OEM Adapter panel.

## Production Notes

This package is UI-model focused. The browser Studio app can consume these panel models to render discovery, diagnostics, signal, sensor, OTA and OEM adapter workflows.
